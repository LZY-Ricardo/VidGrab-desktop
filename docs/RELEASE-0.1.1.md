# VidGrab Desktop 0.1.1

发布日期：2026-04-03

本次版本重点完成了桌面端 AI 分析链路和应用图标的收口。

## 更新内容

- AI 分析改为本地 resolver 优先准备 transcript，再调用云端分析接口
- 无字幕视频增加本地 ASR 转写回退，降低机房风控对解析链路的影响
- 修复重复分析同一视频时临时音频路径识别失败的问题
- 更新桌面端应用图标，使中心图形与网页端品牌符号保持一致

## 产物

- `VidGrab Desktop_0.1.1_x64-setup.exe`
- `VidGrab Desktop_0.1.1_x64_en-US.msi`

## 已知说明

- Windows 安装时可能提示“未知发布者”，当前版本未做代码签名
- 首次执行本地转写时可能需要下载 whisper 模型，耗时会明显高于有字幕视频
